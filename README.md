ECEL Ingeniería y Construcción – Landing Page Corporativa

Landing page corporativa moderna desarrollada para ECEL Ingeniería y Construcción, empresa especializada en ejecución de obras, supervisión de proyectos y soluciones técnicas integrales en el sector construcción.

🎯 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una landing page profesional, elegante y optimizada, orientada a fortalecer la presencia digital de ECEL y comunicar de manera clara sus servicios, experiencia y capacidad técnica.

El sitio está diseñado con un enfoque arquitectónico y corporativo, alineado al rubro de ingeniería y construcción.

🎨 Características
Diseño corporativo, técnico y profesional
Animaciones suaves y elegantes (300–500ms)
Arquitectura basada en componentes reutilizables
Navegación con scroll fluido entre secciones
Diseño responsive (desktop, tablet y móvil)
Botón flotante de WhatsApp siempre visible
Optimización de rendimiento (lazy loading, carga eficiente)
Identidad visual alineada al sector construcción
🛠 Stack Tecnológico
React 18
Vite
Tailwind CSS
Framer Motion
React Icons
React Intersection Observer
📦 Instalación
# Ubicar el proyecto
cd LandingECEL

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build de producción
npm run build

# Previsualizar
npm run preview
