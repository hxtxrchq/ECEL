# Iconos de Redes Sociales

Coloca aquí los iconos/logos de redes sociales en formato PNG o SVG.

## Archivos esperados:

- `whatsapp.png` (20x20px recomendado)
- `facebook.png` (20x20px recomendado)
- `instagram.png` (20x20px recomendado)

## Características:

- Los iconos deben ser cuadrados
- Recomendado: 20x20px a 32x32px
- Formato: PNG con fondo transparente o SVG
- Si no encuentras estos archivos, se mostrarán automáticamente los iconos de React Icons

## Dónde encontrar:

- Puedes descargar de Font Awesome, Feather Icons, o Material Icons
- Asegúrate de que sean a una sola tinta (monocromo) para mejor integración
