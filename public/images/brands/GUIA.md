# Logos de Aliados

Coloca aquí los logos de los aliados/empresas en formato PNG, SVG o JPG.

## Grid: 3 columnas x 3 filas (9 logos total)

## Instrucciones

1. Copia los logos de tus aliados a esta carpeta
2. Nómbralos como: `brand-1.png`, `brand-2.png`, `brand-3.png`... hasta `brand-9.png`
3. Los logos se mostrarán automáticamente en un grid 3x3

## Recomendaciones

- Usa PNG o SVG para mejor calidad
- Tamaño recomendado: 200x120px a 300x180px
- Fondo transparente para mejor integración
- Mantén consistencia en tamaño para mejor presentación
